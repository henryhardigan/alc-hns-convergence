# alc-hns-convergence
A 30-residue segment of bacteriophage T4 Alc converges unusually specifically on the E. coli/Shigella state of H-NS dimerization site 2, a known oligomerization surface required for rigid nucleoprotein filament formation and gene silencing. Proteome-wide motif rarity, H-NS-family restriction, and null-model analyses argue that this pattern is not explained by amino-acid composition alone or by generic similarity to H-NS-family proteins. These observations support a biologically plausible model in which Alc competitively interferes with H-NS filament assembly during T4 infection.
